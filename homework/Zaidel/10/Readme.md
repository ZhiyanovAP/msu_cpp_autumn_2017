# Дз 10
**Ping pong**

Классическая задача.

Два потока по очереди выводят в консоль сообщение. Первый выводит ping, второй выводит pong.



## Сборка
Сборка исходников
```sh
$ make
```
Запуск:
```sh
$ ./result
```

## Вывод:

количество итераций задается в коде программы.  

```
ping
pong
ping
pong
ping
pong
ping
pong
ping
pong
```
