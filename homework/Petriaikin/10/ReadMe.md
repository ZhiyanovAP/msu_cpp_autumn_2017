
# О программе

Использование:  
./PingPong  
Генерирует 10000 словосочетаний "ping pong " (количество задано в define).


# Скрипт  

testscript.sh - скрипт для проверики программы. Выводит  
	0  
	Threading test OK  
если вывод программы содержал только словосочетания "ping pong " (в таком порядке)
