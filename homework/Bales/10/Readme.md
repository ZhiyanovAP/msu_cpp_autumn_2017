# Work_10
__Ping pong__

Классическая задача.

Два потока по очереди выводят в консоль сообщение. Первый выводит **ping**, второй выводит **pong**.

## How run
Осуществляем сборку исходников следующим образом:
```sh
$ make
```
Запускаем:
```sh
$ ./ping-pong
```

## Вывод:
```
ping pong
ping pong
ping pong
ping pong
ping pong
ping pong
ping pong
ping pong
ping pong
ping pong
```
