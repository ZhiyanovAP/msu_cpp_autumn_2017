## Класс-матрица

Шаблон класса-матрицы

### Поддерживаемые операции

    умножение на вектор
    умножение на число
    сравнение матриц на равенство
    сравнение матриц на неравенство
    перемещение std::move()


### Компиляция тестов и их запуск

Из директории исходного файла выполнить

    make
    ./matrixtest

    Matrix<int> m(3, 1);
    m[0][0] = 1;
